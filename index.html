Photica PhotoPulse

If you are looking for Photica's flagship application, PhotoPulse, it is no longer for sale. However the source code to the application has been made available.

PhotoPulse was a commercial Java desktop application from Photica that created animated Flash slideshows from photos and music. It includes a keyframe based pan/zoom editor and a number of photo transitions and effects.

Photica PhotoPulse Source Code

It is being open sourced under the MPL 1.1 license.
